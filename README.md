Iris Flower Classification Using Machine Learning

----Project Description----

This project focuses on identifying the species of Iris flowers using machine learning algorithms. The Iris dataset is a popular dataset in the field of data science and machine learning, commonly used to demonstrate classification techniques.

The model uses four flower measurements:

Sepal Length
Sepal Width
Petal Length
Petal Width

Based on these measurements, it predicts whether the flower belongs to one of the following species:

Iris Setosa
Iris Versicolor
Iris Virginica

----Project Goals----

Load and understand the Iris dataset.
Explore and visualize the data.
Prepare the dataset for machine learning.
Train classification models to recognize flower species.
Measure and compare model performance.
Predict flower species from new input data.
🛠️ Tools and Libraries Used
Python
Pandas (Data Processing)
NumPy (Numerical Computation)
Matplotlib (Data Visualization)
Scikit-learn (Machine Learning)

----Dataset Information----

The project uses the Iris Flower Dataset, which contains measurements of different Iris flowers.

Features
Sepal Length (cm)
Sepal Width (cm)
Petal Length (cm)
Petal Width (cm)
Target Classes
Setosa
Versicolor
Virginica

----Data Preparation----

The following preprocessing steps were performed:

Loaded the dataset using Scikit-learn.
Checked the dataset structure and feature information.
Verified that there were no missing values.
Split the data into training and testing sets.
Prepared the dataset for model training.

----Machine Learning Model----

A Random Forest Classifier was used to train the model.

Workflow
Load the Iris dataset.
Separate features and target labels.
Split the dataset into training and testing data.
Train the Random Forest model.
Make predictions on test data.
Evaluate model accuracy and performance.

----Model Performance----

The trained model achieved high classification accuracy on the test dataset.

Evaluation metrics included:

Accuracy Score
Classification Report
Precision
Recall
F1-Score

The model successfully classified most Iris flower samples with very few errors.

----Result----

This project demonstrates how machine learning can be used to classify Iris flower species based on flower measurements. The Random Forest algorithm provided excellent results and showed the effectiveness of supervised learning for classification tasks.
